ajax

Set off an HTTP request.  Uses CORS, if available on the platform.

Parameters

(given as an Object)

urlURL to send the request to
successcallback function to call when request succeeded
errorcallback function to call when request failed
method(optional) HTTP request method to use (default: GET)
headers(optional) object containing request headers to set
timeout(optional) milliseconds until request is given up and error callback is called.  If omitted, the request never times out.

Example

platform.ajax({
  url: "http://en.wikipedia.org/wiki/AJAX",
  success: function(responseText, responseHeaders) {
    console.log("Here's the page: ", responseText);
  },
  error: function(errorMessage) {
    console.error("Something went wrong: ", errorMessage);
  },
  timeout: 3000
});

Platform support

web browserYES (if browser supports CORS)
IEPartially, no support for setting headers.
nodeYES, CORS not an issue at all
