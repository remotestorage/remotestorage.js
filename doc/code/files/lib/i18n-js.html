Properties

helpers

Object of helpers for i18n/l10n.  Do not cache or extend this property, it gets replaced when the locale changes.

Methods

timeAgo(usec)Example: timeAgo(3000); // -> “3 seconds ago”

defaultLocale

Default locale to use when detection fails.  Initially set to ‘en’.

locales

Array of available locales.

Functions

getLocale

getLocale: function()

Get currently used locale.

setLocale

setLocale: function(locale)

Set locale.

Parameters

localeA string.  Must be included in locales.

Throws <UndefinedLocale> when the locale is unknown.

autoDetect

autoDetect: function()

Automatically determines and sets locale.  This won’t overwrite a stored locale.

detectLocale

detectLocale: function()

Detects current locale from environment.

Sources

(nodejs)process.env.LANG
(browser)navigator.language

Returns

a locale string or undefined.

t

t: function()

Look up a translation.

Parameters

(any number of strings)list of keys to follow in translation tree
variables(optional) Object, containing variables to replae.

Example

// given the translations are as follows:
{
  "a": {
    "b": {
      "c": "Hello {who}!"
    }
  }
}

// Then the following call:
i18n.t('a', 'b', 'c', { who: 'World' });

// returns:
"Hello World!"

clearSettings

Clear cached locale settings.  Doesn’t reset the current in-memory state (i.e. current locale and default locale)
